Introducción

En la actualidad, el avance tecnológico provoca que millones de dispositivos sean reemplazados cada año, aun cuando muchos de ellos siguen siendo funcionales. Este consumo acelerado contribuye al aumento de los residuos electrónicos y a un impacto ambiental cada vez mayor. Por ello, surge la necesidad de buscar alternativas sostenibles que permitan aprovechar al máximo los recursos disponibles y fomentar un modelo de consumo más responsable.
Second Life nace como una respuesta a este problema, ofreciendo una solución basada en la reutilización y el reacondicionamiento de equipos informáticos. Somos un grupo de estudiantes de Sistemas Microinformáticos y Redes comprometidos con el medioambiente y con la accesibilidad a la tecnología. Nuestra meta es dar una segunda oportunidad a dispositivos que aún pueden rendir perfectamente, haciéndolos accesibles a precios asequibles y contribuyendo, al mismo tiempo, a la reducción de residuos electrónicos en nuestra comunidad.

¿Cuál fue nuestra motivación?

Nuestra motivación nace del deseo de aprovechar nuestros conocimientos técnicos para crear un proyecto real que responda a una necesidad social y ambiental. Como estudiantes de informática, somos conscientes de la enorme cantidad de equipos que se tiran a la basura cada día sin haber agotado su vida útil. Esto no solo representa un gasto económico para muchas personas, sino también un daño significativo para el planeta. Por ello, decidimos convertir este problema en una oportunidad para innovar, aprender y aportar una solución beneficiosa para todos.
Además, queríamos asumir un reto que nos permitiera trabajar como un auténtico equipo profesional: planificar, organizar recursos, tomar decisiones y aplicar lo aprendido en el ciclo formativo. Con Second Life buscamos demostrar que es posible emprender desde una visión sostenible, acercando la tecnología a quienes más la necesitan y fomentando un consumo más responsable. Nuestro objetivo es sentirnos orgullosos de crear algo útil, que no solo cumpla con los requisitos académicos, sino que también deje una huella positiva en nuestra comunidad.

Objetivos

Este proyecto transversal se desarrolla en conjunto con el módulo de Empresa e Iniciativa Emprendedora (EIE), y tiene como finalidad trasladar la idea de negocio Second Life al entorno digital mediante una página web creada con HTML, CSS y herramientas de control de versiones como GitHub. A través de este trabajo, no solo buscamos construir una base sólida para nuestra empresa, sino también fortalecer nuestras habilidades técnicas y emprendedoras, cumpliendo con los criterios establecidos en la rúbrica de evaluación.

Los objetivos se estructuran en dos categorías principales:

Objetivos generales

  1. Desarrollar, explicar y presentar de forma clara y detallada nuestro proyecto empresarial Second Life, integrando aspectos de marketing, sostenibilidad y modelo de negocio.
  2. Crear una página web funcional utilizando HTML y CSS que represente profesionalmente la identidad de la empresa.
  3. Aplicar una metodología de trabajo colaborativa y organizada mediante GitHub, gestionando versiones y aportaciones del equipo.

Objetivos especificos

  • Utilizar correctamente el lenguaje HTML para estructurar la página web.
  • Aplicar CSS para el diseño visual, mejorando la presentación y experiencia del usuario.
  • Incluir enlaces, multimedia, navegación y secciones bien definidas dentro del sitio web.
  • Personalizar los elementos visuales para transmitir la imagen corporativa de Second Life.
  • Aprender y utilizar GitHub para el control de versiones y trabajo en equipo dentro del proyecto.
  • Presentar información del proyecto de manera digital, accesible y visualmente atractiva.

Técnologías usadas

HTML5 (HyperText Markup Language)
Lenguaje utilizado para estructurar la página web. Permite definir elementos como textos, imágenes, menús y secciones del sitio.
CSS3 (Cascading Style Sheets)
Tecnología empleada para dar estilo y diseño visual a la web. Con CSS personalizamos colores, tipografías, tamaños, fondos y la apariencia general del sitio.
Git y GitHub
Herramientas que permiten trabajar de manera colaborativa mediante control de versiones. Con GitHub gestionamos el repositorio del proyecto y organizamos los cambios del equipo.
Visual Studio Code
Editor de código utilizado para programar en HTML y CSS de forma eficiente, añadiendo extensiones que facilitan la escritura y el control de Git.
Navegadores Web
Se realizaron pruebas en distintos navegadores como Chrome, Firefox o Edge, para asegurar el correcto funcionamiento y compatibilidad del sitio web.
